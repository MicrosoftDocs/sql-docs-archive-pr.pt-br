# <a name="trace-file-properties-general-tab"></a>Propriedades do Arquivo de Rastreamento (guia Geral)
  Use a guia **Geral** da caixa de diálogo **Propriedades do Arquivo de Rastreamento** para exibir as propriedades de um arquivo de rastreamento.  
  
 Para exibir essa janela, abra um arquivo de rastreamento. Em seguida, no menu **Arquivo** , clique em **Propriedades**.  
  
## <a name="options"></a>Opções  
 **Nome do arquivo**  
 O caminho e o nome do arquivo de rastreamento exibido.  
  
 **Nome do provedor de rastreamento**  
 Exibe o nome da instância do [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] que foi rastreada.  
  
 **Tipo de provedor de rastreamento**  
 Mostra o tipo de servidor que forneceu o rastreamento.  
  
 **version**  
 Exibe a versão do servidor que forneceu o rastreamento.  
  
 **Tamanho do arquivo (KB)**  
 O tamanho do arquivo de rastreamento em kilobyte (KB).  
  
 **Criado**  
 A data e hora em que o arquivo de rastreamento foi criado.  
  
 **Alteração**  
 A data e hora em que o arquivo de rastreamento foi modificado.
