 <span data-ttu-id="8cd22-109">Para conceder acesso a conjuntos de dados compartilhados, você pode permitir que os usuários herdem as atribuições de função existentes de uma pasta pai ou criem uma nova atribuição de função no próprio item.</span><span class="sxs-lookup"><span data-stu-id="8cd22-109">To grant access to shared datasets, you can allow users to inherit existing role assignments from a parent folder or create a new role assignment on the item itself.</span></span>

 <span data-ttu-id="8cd22-113">Para fornecer mais controle sobre conjuntos de dados compartilhados e os resultados das respectivas consultas, configure a segurança em nível de item no item do conjunto de dados compartilhado ou salve conjuntos de dados compartilhados em uma pasta e configure a segurança em nível de item na pasta.</span><span class="sxs-lookup"><span data-stu-id="8cd22-113">To provide more control over shared datasets and their query results, configure item level security on the shared dataset item or save shared datasets to a folder and configure item level security on the folder.</span></span>
