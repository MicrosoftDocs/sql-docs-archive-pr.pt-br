# <a name="virtualdirectoryreportserver-property-wmi-msreportserver_configurationsetting"></a>Propriedade VirtualDirectoryReportServer (WMI MSReportServer_ConfigurationSetting)
  Retorna o diretório virtual para o aplicativo do servidor de relatório. Somente leitura.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
Public Dim VirtualDirectoryReportServer As String  
```  
  
```csharp  
public string VirtualDirectoryReportServer;  
```  
  
## <a name="property-values"></a>Valores da propriedade  
 Um `string` que contém o diretório virtual.
