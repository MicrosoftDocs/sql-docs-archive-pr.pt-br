 <span data-ttu-id="ffc69-117">Além de usar a autenticação e a autorização para controlar o acesso aos dados e objetos replicados, a replicação inclui duas opções para controlar quais dados estão disponíveis no Assinante: a filtragem de colunas e a filtragem de linhas.</span><span class="sxs-lookup"><span data-stu-id="ffc69-117">In addition to using authentication and authorization to control access to replicated data and objects, replication includes two options to control what data is available at a Subscriber: column filtering and row filtering.</span></span> <span data-ttu-id="ffc69-118">Para mais informações sobre filtragem, consulte [Filtrar dados publicados](../publish/filter-published-data.md).</span><span class="sxs-lookup"><span data-stu-id="ffc69-118">For more information about filtering, see [Filter Published Data](../publish/filter-published-data.md).</span></span>
