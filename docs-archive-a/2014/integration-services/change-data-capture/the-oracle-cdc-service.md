## <a name="security-considerations"></a><span data-ttu-id="d5cef-129">Considerações de segurança</span><span class="sxs-lookup"><span data-stu-id="d5cef-129">Security Considerations</span></span>

### <a name="protection-of-source-oracle-data"></a><span data-ttu-id="d5cef-131">Proteção dos dados do Oracle de origem</span><span class="sxs-lookup"><span data-stu-id="d5cef-131">Protection of Source Oracle Data</span></span>

data-ttu-id="d5cef-137">Os dados capturados estão armazenados em instâncias de captura do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] e oferecem a mesma proteção fornecida para
